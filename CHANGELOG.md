## [1.0.0] – 2026

### Added

- Bootstrap-style layout (navbar, hero, sections) without Bootstrap dependency
- Home page: hero, get started, demos (flip card, cube, carousel, tiles), customize
- Gallery page: book stack, door, carousel, floating cards, 3D text, frames, accordion, ribbon
- Experiments page: bent panel, rotating text, card reveal, folding card, stacked layers
- Showcase page: 35+ 3D effects (coin flip, card lift, revolving door, book open, zigzag, neon, pulse circle, shutter, fan out, peel, layered cards, and more)
- Customizer: accent color, animation speed, border radius, theme (Auto/Light/Dark), persisted in localStorage
- Portable dist: `dist/3d-gallery.css`, `dist/3d-gallery.js`, `dist/3d-gallery.min.css`, `dist/3d-gallery.min.js`
- Minified production assets (clean-css, terser) — use like Bootstrap’s compiled/minified files
- Downloads page: compiled vs minified, source files, include-in-project examples, data attributes
- Documentation: README.md, USAGE.md, CUSTOMIZE.md, CHANGELOG.md, CONTRIBUTING.md, LICENSE.md
- Copy-to-clipboard on all code blocks
- Output-first then code pattern on all demo sections
- Responsive breakpoints (576px, 768px, 992px) and prefers-reduced-motion support
- Theme toggle, back to top, smooth scroll
