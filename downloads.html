Compiled CSS and JS

Ready-to-use compiled code for 3D Transform Gallery (rskworld.in). Like Bootstrap: choose development (readable) or production (minified).

  • CSS — flip cards, cube, carousel, tiles, responsive layout
  • JS — theme toggle, back to top, smooth scroll, customizer

No Bootstrap dependency. All styles are scoped so they don’t conflict with your site.

Compiled (development): 3d-gallery.css 3d-gallery.js
Minified (production, like Bootstrap): 3d-gallery.min.css 3d-gallery.min.js

Source files

Use the full project to get all source CSS (variables, theme, components, responsive) and HTML demos. Copy the entire project folder or clone the repo. No build step required for the pre-built dist/ files.

  • dist/3d-gallery.css / dist/3d-gallery.min.css — portable 3D styles (minified for production)
  • dist/3d-gallery.js / dist/3d-gallery.min.js — portable JS (minified for production)
  • css/ — full source (variables, bootstrap-replica, theme, components, etc.)
  • index.html, gallery.html, experiments.html, showcase.html — demo pages

Include in your project

Copy the dist folder to your project, then add the CSS and JS. Wrap 3D content in class="rsk-3d-gallery" so styles stay scoped.

Output

Result: flip card with link + script

Front
Back

Code

HTML
<!-- Production: use .min.css and .min.js (compressed like Bootstrap) -->
<link rel="stylesheet" href="path/to/dist/3d-gallery.min.css">
<!-- Your content -->
<div class="rsk-3d-gallery">
  <div class="flip-cards">
    <div class="flip-card">
      <div class="flip-inner">
        <div class="flip-front">Front</div>
        <div class="flip-back">Back</div>
      </div>
    </div>
  </div>
</div>
<script src="path/to/dist/3d-gallery.min.js"></script>

Optional: theme toggle and back to top

Add buttons that work with the included script:

HTML
<button type="button" data-3d-theme-toggle aria-label="Toggle theme">&#9728;</button>
<button type="button" data-3d-back-to-top aria-label="Back to top">&#8593;</button>

Data attributes (JS)

Supported by dist/3d-gallery.js (rskworld.in):

Attribute Purpose
data-3d-theme-toggleButton to switch light/dark (sets data-theme on <html>)
data-3d-back-to-topShows on scroll, scrolls to top
data-3d-customizer-toggleOpens customizer panel
data-3d-customizer-overlayOverlay for customizer
data-3d-customizer-panelCustomizer panel container
data-3d-customizer-closeClose button inside panel
