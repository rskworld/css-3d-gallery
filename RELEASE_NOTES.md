## v1.0.0 (2026)

**First stable release.** Pure CSS 3D transformations with a Bootstrap-style UI (no Bootstrap dependency). By [rskworld.in](https://rskworld.in).

### Highlights

- **Bootstrap-style layout** — Navbar, hero, sections; no Bootstrap library used
- **4 demo pages** — Home, Gallery (More), Experiments, Showcase (35+ effects)
- **Portable dist** — `3d-gallery.css`, `3d-gallery.js` + minified (`.min.css`, `.min.js`) for production
- **Customizer** — Accent color, animation speed, border radius, theme (Auto/Light/Dark), persisted in localStorage
- **Documentation** — README, USAGE, CUSTOMIZE, CHANGELOG, CONTRIBUTING, LICENSE
