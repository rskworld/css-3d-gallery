Customize with CSS variables

Override accent, background, and 3D sizing with CSS variables. Use the customizer panel (gear icon) on this site or set data-theme="light" for light mode.

Code

CSS
.rsk-3d-gallery {
  --g3d-accent: #0d6efd;
  --g3d-cube-size: 120px;
  --g3d-duration: 0.4s;
}
