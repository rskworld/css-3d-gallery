# Contributing to 3D Transform Gallery

<!-- Author: Molla Samser | rskworld.in | 2026 -->

Thank you for your interest in **3D Transform Gallery**. This project is maintained by [rskworld.in](https://rskworld.in).

---

## How to contribute

1. **Use the project** — Try the demos, use the dist files on your site, and share feedback.
2. **Report issues** — If you find a bug or have a suggestion, contact us:
   - Email: help@rskworld.in | support@rskworld.in
   - Website: [rskworld.in](https://rskworld.in)
3. **Suggest 3D effects** — Ideas for new CSS 3D effects or layout improvements are welcome.
4. **Documentation** — Fixes or improvements to README.md, USAGE.md, or CUSTOMIZE.md are appreciated.

---

## Project structure

- **css/** — Source styles (variables, theme, components, features)
- **dist/** — Portable CSS/JS (and .min versions) for use on any site
- **js/** — Customizer and utilities
- **\*.html** — Demo pages (index, gallery, experiments, showcase, downloads)
- **\*.md** — Documentation (README, USAGE, CUSTOMIZE, CHANGELOG, CONTRIBUTING, LICENSE)

---

## Code style

- Use existing CSS variables and class names where possible.
- Keep 3D effects scoped under `.rsk-3d-gallery` in dist files.
- Test in a modern browser and check reduced motion (accessibility).
